# Shared Templates

Shared templates used across multiple factory workers.

## Structure

### `durable-objects/`
Durable Object templates:
- `hello-world/` - Basic Durable Object example
- `chat/` - Chat application using Durable Objects

### `actors/`
Cloudflare Actors framework templates and examples.

### `workflows/`
Workflow templates for orchestrating multi-step processes.

### `queues/`
Queue templates:
- `pipelines/` - Data pipeline queue templates

## Usage

These templates are shared across factories and can be referenced by any worker that needs Durable Objects, Actors, Workflows, or Queues functionality.
