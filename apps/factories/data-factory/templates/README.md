# Data Factory Templates

Templates for working with Cloudflare data storage services.

## Templates

### `r2-explorer/`
R2 object storage explorer template for browsing and managing R2 buckets.

### `d1-template/`
D1 database template with migrations and basic CRUD operations.

### `d1-explorer/`
D1 database explorer (to be added).

### `r2-data-catalog/`
R2 data catalog examples for organizing and cataloging R2 data:
- PySpark notebook examples
- Spark Scala examples

### `kv-handling/`
KV (Key-Value) storage templates:
- `to-do-list/` - Example KV-based todo list application

### `pipelines/`
Data pipeline templates for processing data workflows.

## Usage

Each template includes:
- Setup instructions
- Example code
- Migration scripts (where applicable)
- README with specific usage
