# Services Factory Templates

Templates for building service workers and utilities.

## Templates

### `queues/`
Cloudflare Queues templates (to be added).

### `web-crawler/`
Web crawling service template using Cloudflare Queues for distributed crawling.

### `browser-render/`
Browser automation and rendering templates:

#### `playwright/`
- `tests/` - Playwright testing templates

#### `puppeteer/`
Puppeteer-based browser automation (to be added).

#### `utilities/`
Browser utility templates:
- `tojson/` - Convert pages to JSON
- `tomarkdown/` - Convert pages to Markdown
- `extract-text/` - Extract text content
- `extract-html/` - Extract HTML content
- `convert-pdf/` - PDF conversion utilities
- `run-testing/` - Browser testing utilities

## Usage

Each template includes setup instructions and example implementations.
