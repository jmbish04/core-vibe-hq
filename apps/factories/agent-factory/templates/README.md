# Agent Factory Templates

This directory contains templates for building AI agents using various frameworks and patterns.

## Structure

### `cloudflare-agents-sdk/`
Templates and examples using Cloudflare's Agents SDK:
- `agents-starter/` - Starter template for Agents SDK
- `examples/` - Example implementations from Cloudflare
- `guides/` - Guide implementations
- `packages/` - Agents SDK packages
- `actors/` - Cloudflare Actors framework examples
- `durable-objects/` - Durable Object examples for agents

### `openai-sdk/`
Templates using OpenAI SDK:
- Various OpenAI SDK examples and patterns

### `ai-sdk/`
Templates using Vercel AI SDK:
- `demos/` - AI SDK demo patterns
- `libs/` - AI SDK libraries
- `packages/` - AI SDK packages
- `types/` - TypeScript types

### `patterns/`
Reusable agent patterns:
- `vectorize/` - Vectorize patterns
- `embeddings/` - Embedding patterns
- `autorag/` - AutoRAG patterns
- `rag/` - RAG (Retrieval Augmented Generation) patterns

### `demos/`
Agentic pattern demonstrations from STAGING/ai/demos:
- Agent schedulers
- Task managers
- Human-in-the-loop patterns
- MCP (Model Context Protocol) examples
- Tool calling patterns
- Structured output patterns
- And more...

## Usage

Each template directory contains its own README with specific usage instructions.

## Organization by Framework

- **Cloudflare Agents SDK**: Native Cloudflare agent framework
- **AI SDK**: Vercel AI SDK patterns
- **OpenAI SDK**: Direct OpenAI SDK usage
- **Gemini GenAI**: Google Gemini patterns (to be added)
