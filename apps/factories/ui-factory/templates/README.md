# UI Factory Templates

Frontend worker templates organized by framework and implementation.

## Structure

### `demos/react/`
React-based templates:
- `postgres-fullstack/` - React with Postgres fullstack
- `c-code-react-runner/` - React code runner template

### `demos/nextjs/`
Next.js templates:
- `starter/` - Next.js starter template
- `c-code-next-runner/` - Next.js code runner template

### `demos/remix/`
Remix templates:
- `starter/` - Remix starter template

### `demos/vite/`
Vite-based templates:
- `react/` - Vite + React template
- `vite-cf-DO-runner/` - Vite with Durable Objects
- `vite-cf-DO-v2-runner/` - Vite with Durable Objects v2
- `vite-cf-DO-KV-runner/` - Vite with Durable Objects and KV
- `vite-cfagents-runner/` - Vite with Cloudflare Agents SDK

### `demos/react-router/`
React Router templates:
- `starter/` - React Router starter
- `hono-fullstack/` - React Router with Hono backend
- `postgres-ssr/` - React Router with Postgres SSR

## Organization

Templates are organized by:
1. **Framework**: react, nextjs, remix, etc.
2. **Implementation**: vite, react-router, etc.

Each template includes:
- Setup instructions
- Example components
- Configuration files
- README with specific usage
